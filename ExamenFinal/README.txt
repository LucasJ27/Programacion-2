Documentación de API para Postman
A continuación se detallan las URLs y métodos HTTP disponibles para interactuar con la API mediante Postman.

Cliente
GET ALL: /cliente
GET BY ID: /cliente/{id}
POST: /cliente
-> Body: Cliente
PUT: /cliente
-> Body: Cliente
DELETE: /cliente/{id}
